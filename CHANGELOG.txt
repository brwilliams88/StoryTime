StoryTime — Change History
==========================
Versioning guide:
  v0.x   = Pre-release / building toward MVP
  v1.0   = First fully working end-to-end app (generate, read, save, cross-device)
  v1.x   = Feature additions and polish after MVP
  v2.0   = Major redesign or architectural overhaul (rare)
  Minor rev (+0.1): new feature, meaningful UI change, new integration
  Patch rev (+0.0.1): bug fix, small tweak (optional — skip if change is trivial)

--------------------------
v0.4 — 2026-05-23 — Form Simplification + Prompt Quality
  • Form reduced to 7 fields, new order: Characters → Anything Else → Age → Length → Genre → Mood → Theme
  • Characters now free-text (any description or named characters)
  • Anything Else merges old Setting + Special Touches into one box
  • New Mood multi-select (Funny, Surprise twist, Heartwarming, Action-packed, Dreamy)
  • New Theme field: 10 preset pills + custom text input ("the moral of the story is...")
  • Genres rebuilt: Adventure, Fairy Tale, Fantasy, Sci-Fi, Pirates, Superhero, Mystery, Friendship, Spooky
  • Removed: Bedtime/Calming genre, character-type dropdown, separate Setting/Lesson fields, Artwork Style selector (returns with image gen)
  • Length minutes updated: 2 / 5 / 8 / 12
  • Prompt rewritten: quality directives first (cohesion, arc, consistent characters), structure second
  • Auto-retry once silently on malformed JSON
  • All selectors now pill-style (no native HTML dropdowns)
  • Removed emoji from action buttons (kept on content tags)
  • Debug panel: app version + build date displayed
  • Loading hint scales by length
