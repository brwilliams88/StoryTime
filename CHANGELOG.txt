StoryTime — Change History
==========================
Versioning guide:
  v0.x   = Pre-release / building toward MVP
  v1.0   = First fully working end-to-end app (generate, read, save, cross-device)
  v1.x   = Feature additions and polish after MVP
  v2.0   = Major redesign or architectural overhaul (rare)
  Minor rev (+0.1): new feature, meaningful UI change, new integration
  Patch rev (+0.0.1): bug fix, small tweak (optional — skip if change is trivial)

--------------------------
v0.9.21 — 2026-06-25 — Fix stuck close-swipe on phone; looser swipe-to-turn
  • FIXED the freeze: swiping from page 1 back to the cover on a phone removed the page element mid-swipe, which stops iOS touch events and locked up the reader (couldn't turn pages or tap back). The close is now played as a triggered animation (swipe back → it plays + settles to centre) instead of finger-following, so the page element is never yanked out from under your finger. Added an "ignore touch while a turn is animating" guard too.
  • Dropped the strict swipe ZONES. Now any directional swipe turns the page (forward = swipe toward the spine, back = swipe outward), and a natural grab-and-drag of a page still finger-follows. Nothing "does nothing" anymore. Turn speed unchanged (slow enough to see flicks).

--------------------------
v0.9.20 — 2026-06-25 — Grab-the-page swipes, finger-following fixed, slower turns, glossy cover
  • SWIPE ZONES: page turns now respond to which page you "grab" — swipe the RIGHT page (landscape) / BOTTOM page (portrait) to go forward, the LEFT/TOP page to go back. Swiping the wrong page or wrong direction does nothing. More book-like, and it makes finger-following reliable.
  • FIXED page 1 → 2 not finger-following: the cover-close interception on page 1 was forcing flick-only turns. Now only the cover itself is intercepted; every interior turn (and the close) goes straight to the finger-following engine.
  • Close the book (page 1 → cover) is now a normal finger-following back-turn that reveals the book and settles it to centre (via new setIndex/afterTurn hooks in the turn engine).
  • SLOWER turns so you can actually watch them: page-turn commit 300ms → 520ms; cover open/close slide 0.36s → 0.5s.
  • COVER realism: gently rounded corners + a glossy diagonal sheen, matching the bookshelf books (which use a gloss highlight + spine ridge + rounded corners + shadow).

v0.9.19 — 2026-06-25 — Cover open reuses the real page-turn; thicker scaling spine
  • Spine on the cover is now ~2.5x thicker and scales with the book (6.5% of the book's short side) instead of a fixed pixel size, so it stays proportional as the image grows.
  • Cover open/close reworked to REUSE the normal page-turn engine: the closed book first slides into its hinge-side half (down in portrait, right in landscape) while only the book + dark stage show (no early peek at the spread), then pageCurl turns it exactly like an interior page — the cover lifts and the first spread's half lays down on the other side. Close runs the same turn in reverse, then slides the book back to centre. This makes the open/close identical in feel to regular page turns and fixes the landscape "next page shows before it opens" bug.

  • Cover redesigned to match the bookshelf book: cover art + cream title plate + a small "By …" byline, with a spine on the HINGE edge and no page-edge stack / no shelf lean. It's sized to exactly ONE interior page and centred.
  • Open animation: on turn the book slides so its hinge meets the centre crease (portrait: down so the top edge is centred; landscape: right so the left edge is centred), then the front swings open about that hinge to reveal the first spread — same feel as the interior page turns, just with the slide first.
  • Added the mirror CLOSE animation: going back from page 1 to the cover swings the book shut and slides it back to centre.
  • Notes: rotate signs (portrait/landscape) may need a flip on device; turning from page 1 → 2 now plays as a triggered turn (not finger-following) because the cover gestures are intercepted there — easy to revisit.
