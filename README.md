# StoryTime 📖

A personal AI-powered bedtime story generator for Kai and Nozomi.

Built as a Progressive Web App (PWA) — installable on iPhone and iPad.
Generates illustrated stories using OpenAI GPT-4o (text) and DALL-E 3 (images).

## Tech Stack
- **Frontend:** Vue.js (CDN), HTML, CSS
- **Hosting:** GitHub Pages
- **Database & Storage:** Supabase (PostgreSQL + file storage)
- **AI:** OpenAI API (GPT-4o + DALL-E 3)
- **API Proxy:** Cloudflare Workers (keeps API key out of browser)
- **Narration:** Web Speech API (built into device browsers)
